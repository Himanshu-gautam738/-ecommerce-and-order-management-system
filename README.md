# Ecommerce and Order Management System

## Project Overview

This is a full-stack Ecommerce and Order Management System developed using React, Django, and Django REST Framework.

The project provides user authentication, email verification, product management, cart functionality, checkout process, and order tracking features.

---

## Features

* User Signup & Login
* Email Verification
* JWT Authentication
* Product Listing
* Product Details Page
* Add to Cart
* Checkout System
* Orders Page
* Protected Routes
* REST API Integration
* Responsive UI

---

## Tech Stack

### Frontend

* React.js
* Vite
* CSS

### Backend

* Django
* Django REST Framework
* SQLite

---

## Installation

### Backend Setup

```bash
cd backend

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```

### Frontend Setup

```bash
cd frontend

npm install

npm run dev
```
